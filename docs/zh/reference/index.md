本章节提供了 MinerU 项目的详细参考资料。在这里您可以找到技术规范、API 文档、输出文件格式说明以及版本历史记录。

## 文档概览

### 模块功能总览

模块总览文档主要帮助开发者理解：

- **包职责划分**：`cli`、`backend`、`model`、`data`、`utils` 分别负责什么
- **代码阅读入口**：针对不同功能应该先看哪些文件
- **边界划分**：哪些逻辑应该放在编排层、解析层、模型层或存储层

### 输出文件说明

理解 MinerU 生成的输出文件对于有效使用工具至关重要。输出文件文档提供了：

- **可视化调试文件**：帮助您理解文档解析过程
- **结构化数据文件**：包含详细的解析结果，可用于进一步处理
- **文件格式规范**：每种输出文件类型的详细说明

### 文件处理流程

处理流程文档主要说明：

- **请求入口**：上传文件如何被保存和规范化
- **分流规则**：office、PDF、图片如何进入不同处理路径
- **后端执行**：`pipeline`、`vlm`、`hybrid` 的差异
- **结果生成**：Markdown、JSON、图片等文件如何输出

### 更新日志

更新日志记录了 MinerU 的演进历程，包括：

- **版本更新**：每个版本的新功能和改进
- **错误修复**：每个版本中解决的问题
- **重大变更**：可能影响您使用的重要变更
- **功能弃用**：正在逐步淘汰的功能
